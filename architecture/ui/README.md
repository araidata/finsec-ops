# UI Architecture

The approved visual direction is a dark enterprise cybersecurity financial
operations shell:

- Near-black page background
- Navy and graphite panels
- Thin cool borders
- Cyan primary actions and active states
- Teal and blue for healthy budget and forecast signals
- Amber for exposure or pending review
- Green for approved or active status
- Muted red for exception states

Keep layouts dense, table-driven, and operational. Avoid marketing-page
patterns and generic accounting aesthetics.
